    Tasks Action#

    Schedule and manage farming activities, set reminders, and track completion of farm work.

    Purpose#

    Plan and track activities like irrigation, spraying, harvesting, vaccinations, and maintenance. Stay organized with due dates, priorities, and status tracking.

    Key Features#

    • Due dates - Set deadlines for activities
    • Priority levels - Mark tasks as high, medium, or low priority
    • Status tracking - Track pending, in-progress, and completed tasks
    • Reminders - Get notified about upcoming activities
    • Module linking - Tasks are linked to specific crops, livestock, fields, etc.

    Available In#

    Module Example Tasks
    Farm Property maintenance, team meetings, license renewals
    Crops Irrigation, spraying, weeding, fertilizing, harvesting
    Fields Land preparation, boundary repairs, soil testing
    Livestock Vaccination, health checkups, breeding, deworming
    Machines Oil change, servicing, tire replacement, repairs
    Inventory Stock check, reorder supplies, expiry monitoring

    Task Status#

    Status Description
    Pending Task is scheduled but not yet started
    In Progress Work has begun on the task
    Completed Task has been finished

    Priority Levels#

    Priority Use For
    High Urgent tasks that need immediate attention (e.g., pest outbreak, sick animal)
    Medium Important tasks with flexible timing (e.g., scheduled spraying, routine checkup)
    Low Tasks that can be done when time permits (e.g., equipment cleaning, record updates)

    Common Task Examples#

    Crop Tasks#

    • Schedule irrigation every 3 days
    • Spray pesticide next Monday
    • Harvest tomatoes by end of week
    • Apply fertilizer after rain

    Livestock Tasks#

    • Vaccinate cattle on 15th
    • Call vet for health checkup
    • Order feed before stock runs out
    • Clean shed every Saturday

    Machine Tasks#

    • Service tractor after 100 hours
    • Change oil next month
    • Replace worn tires
    • Check hydraulic fluid levels

    Tips#

    • Create tasks for recurring activities like weekly spraying or monthly maintenance
    • Set priorities to focus on urgent farming activities first
    • Review pending tasks daily to stay on top of farm work
    • Link tasks to the correct module for organized tracking
    • Use task descriptions to note specific instructions or quantities

    Best Practices#

    • Plan tasks at the beginning of each week
    • Set realistic due dates based on weather and resource availability
    • Update task status promptly to track actual vs planned work
    • Review completed tasks to understand work patterns
    • Create tasks immediately when you notice something needs attention

    Frequently Asked Questions#

    How do I create a recurring task?

    Currently, each task is created individually. For recurring activities, create a new task after completing the previous one, or plan multiple tasks in advance with different due dates.

    Can I assign tasks to team members?

    Tasks are linked to modules rather than specific team members. All team members with access to that module can see and update the task status.

    What happens to overdue tasks?

    Overdue tasks remain in the pending list until you mark them complete or update the due date. Review overdue tasks regularly to reschedule or complete them.

    Can I see all tasks across modules?

    Yes, from the home screen you can view all tasks across your farm, or filter by specific modules to see only relevant tasks.
