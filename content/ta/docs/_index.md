# பண்ணை ஆவணங்கள்

பண்ணை செயலியைப் பயன்படுத்துவதற்கான முழு வழிகாட்டி.

## பிரிவுகள்

- [தொடங்குதல்](/ta/docs/getting-started/) - செயலி அமைப்பு மற்றும் ஆன்போர்டிங்
- [தொகுதிகள்](/ta/docs/modules/) - பண்ணை, பயிர்கள், கால்நடைகள், மேலும்
- [செயல்கள்](/ta/docs/actions/) - வருமானம், செலவுகள், பணிகள், மேலும்
- [அடிக்கடி கேட்கப்படும் கேள்விகள்](/ta/docs/faq/) - பொதுவான கேள்விகள்
