## Modules

Core modules for managing your farm:

| Module | Description |
|--------|-------------|
| [Farm](/en/docs/modules/farm/) | Farm profile, team management, farm-level activities |
| [Crops](/en/docs/modules/crops/) | Track crops from planting to harvest |
| [↳ Seasons](/en/docs/modules/crops/seasons/) | Organize crop activities by growth cycle |
| [Fields](/en/docs/modules/fields/) | Manage fields and crop rotation history |
| [Livestock](/en/docs/modules/livestock/) | Cattle, poultry, goat, sheep, pig management |
| [Machines](/en/docs/modules/machines/) | Tractors, implements, usage and maintenance |
| [Inventory](/en/docs/modules/inventory/) | Seeds, fertilizers, fuel, spare parts, yields |
| [Contacts](/en/docs/modules/contacts/) | Vendors, buyers, transaction history |
| [Roles](/en/docs/modules/roles/) | Owner, Manager, Worker access levels |

## Actions

Activities you perform on modules:

| Action | Description | Available In |
|--------|-------------|--------------|
| [Income](/en/docs/actions/income/) | Record sales and revenue | Crops, Livestock |
| [Expenses](/en/docs/actions/expenses/) | Track costs and expenditures | All modules |
| [Tasks](/en/docs/actions/tasks/) | Schedule and manage activities | All modules |
| [Notes](/en/docs/actions/notes/) | Record observations and comments | All modules |
| [Yields](/en/docs/actions/yields/) | Track harvest and production | Crops, Livestock |
| [Documents](/en/docs/actions/documents/) | Attach files, photos, receipts | All modules |

### Crop Management

- [Seasons](/en/docs/modules/crops/seasons/) - Organize by growth cycle

- [Animal Types](/en/docs/modules/livestock/#animal-types--options) - Cattle, Poultry, Goat, Sheep, Pig
